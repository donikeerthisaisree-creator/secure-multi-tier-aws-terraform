# Secure Multi-Tier Web Application (AWS + Linux)

## Project Overview
Designed and deployed a secure multi-tier web application using AWS and Linux.

## Services Used
- EC2
- VPC
- Subnets
- Security Groups
- NAT Gateway
- Internet Gateway
- Load Balancer
- Auto Scaling
- CloudWatch
- Terraform

## Features
- Linux-based web server deployment
- Secure networking using VPC
- High availability using Auto Scaling
- Monitoring using CloudWatch
- Infrastructure as Code using Terraform

## Commands
terraform init
terraform plan
terraform apply
terraform destroy
